# 🎮 CodeAlpha Hangman Game

A fun and interactive Hangman word-guessing game built for the **CodeAlpha Python Internship — Task 1**.

> > **Note:**
> The original task required a basic console-based Hangman game.
> To further enhance my learning experience and web development skills,
> I implemented the project as a Flask web application while preserving
> all the core game logic and requirements specified in the task.

## ✅ Requirements Covered

- 5 predefined words ✅
- Max 6 wrong guesses ✅
- Letter by letter guessing ✅
- Win / Lose condition ✅
- Random word selection ✅
- Key concepts: random, if-else, strings, lists ✅

## 🌟 Bonus Features

- 🌐 Flask Web Application
- 🎨 Professional dark theme UI
- 🖼️ SVG Hangman drawing (step by step)
- 💡 Hint system for each word
- ❤️ Lives system
- ❌ Wrong letters tracker
- 📱 Fully responsive design

## 🛠️ Tech Stack

Python · Flask · HTML5 · CSS3 · Bootstrap 5 · Jinja2

## ▶️ How to Run

```bash
pip install flask
python app.py
```

Open: `http://127.0.0.1:5000`

## 🎮 How to Play

1. Click **Start Game**
2. A secret word is randomly chosen
3. Use the on-screen keyboard to guess letters
4. You have **6 wrong guesses** maximum
5. Reveal the full word to **WIN!** 🏆
